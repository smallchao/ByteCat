web开发之实现JWT认证

更新于 2021-09-22
JSON Web Token（JWT）是一个开放的标准，它定义了一个紧凑且自包含的方式，用于在各方之间作为JSON对象安全地传输信息。JWT常常被用作前后端分离场景下的用户验证，一旦用户登录，那么每个后续的请求都将包括JWT，从而允许用户访问该令牌允许的路由、服务和资源。
